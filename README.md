# webscraping 
This repo is about how to take the screenshot of a website using puppeteer.

## Run Locally

Clone the project

```bash

  git clone https://github.com/B-chandru/webscraping.git

```

Go to the project directory

```bash

  cd webscraping

```

Install dependencies

```bash

  npm install

```

Start the server

```bash

  npm run start

```
